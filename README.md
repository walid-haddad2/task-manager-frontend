<h1 align='center'> Task-Manager-Frontend </h1>
This is the frontend UI of the Task Manager application, built with Angular.
Users can register, log in, and manage their tasks through a fast and intuitive interface that communicates with the Spring Boot backend via REST APIs.  
<br>
<br>
The goal of this project is to demonstrate modern Angular development practices, including:  

* Component-based architecture 

* Reactive forms

* State management with RxJS

* Token-based authentication

* Clean UI design with Angular Material
